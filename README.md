# MoodBot Python 
A beginner-friendly Python app that takes your current mood and returns a motivational quote + a daily goal to keep you moving forward.

MoodBot is simple by design. Type in how you're feeling, and MoodBot meets you where you're at, responding with an encouraging quote and a actionable goal tailored to your mood. No fluff, just good vibes and direction.

This project is coded primarily in Python. This project is not done yet, we will soon be adding an html coded frontend to excecute a webpage for MoodBot!

Tech Stack : Python, HTML/CSS (frontend coming soon)

Run this program in Terminal : pip install -r requirements.txt
